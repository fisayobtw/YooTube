## 🚀 Features

- 🔐 Fake login form that captures:
  - Email
  - Password
  - IP address
- 🖱️ JavaScript keylogger that logs all keystrokes in real-time

- 🔁 Redirects to real YouTube after "login"
- 💾 Logs credentials and keystrokes locally
